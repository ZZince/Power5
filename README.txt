# Power5 - Jeu de Stratégie

Power5 est un jeu de stratégie conçu pour 2 à 4 joueurs. Dans ce jeu, chaque joueur dispose de 7 points de mouvement à utiliser chaque tour pour déplacer leurs pièces à travers le plateau. Le but du jeu est de capturer les cellules du  plateau tout en préservant les cellules déjà capturées.

## Comment Jouer

Le jeu se joue sur un plateau de jeu carré avec une grille de cases en 10 par 10. Chaque joueur commence avec 4 pièces  placeer sur les cellules du plateau. Chaque tour, un joueur peut dépenser jusqu'à 7 points de mouvement pour déplacer ses pièces. Les joueurs peuvent se déplacer horizontalement ou verticalement.

Le plateau est constitué de plusieurs types de cases:
	- Normal: Aucun effft particulier
	- Lave: Il est impossible d'aller dans la lave
	- Sable: Le joueur peut passer uniquement une fois dessus par tour
	- Trou Noir: Téléporte le joueur aléatoirement dans un trou blanc
	- Trou Blanc: Il est impossible de rentrer dans un trou blanc, mais ils servent de sortit à un trou noir

Des objets apparaissent aléatoirement sur les cellules du plateau:
	- GameBoardTurner: Retourne le plateau 
	- PieceClearer: Réinitialise toutes les piéces déposées ce tour
	- RunFastet: Donne 3 points de mouvements supplémentaires au joueur
	- CoinsExpensiver: Donne 3 piéces supplémentaires au joueur

## Comment Gagner

Le joueur réussissant à capturer 5 cellules alignées et côte à côte gagne la partie.

## Commandes du Jeu

- Utilisez les touches ZQSD pour se déplacer sur le plateau
- Appuyez sur la touche E pour déposer une piéces
- Appuyez sur F pour terminer votre tour

## Installation du jeu

	L'installation est automatique grâce à l'installer intégré
